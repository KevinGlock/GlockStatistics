Was ist Datenmanagement?

Daten für ein computergestütztes Programm lesbar machen. Wie sollen vorliegende Daten codiert werden? Code Plan bzw. Code Schema bzw.Codebuch Wie sollen fehlende Werte codiert werden? Wie sollen einzelne Variablen im Datensatz beschriftet werden? Verschiedene Dateiformate, je nach Programm und Auswertungsziele .dat, .txt , .sav, …

Wie managed man Daten?

Storage, relationale und nicht relationale databases, SQL, noSQL, accountability Originaldokumente Sicherheitskopien Status der Daten (Metadaten) Aktualisierung von Dateiformaten
